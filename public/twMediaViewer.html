About

Twitter Media Viewer is a JavaScript program for viewing tweets from zip archives created with the twMediaDownloader browser extension.

How to

Simply select a zip created with twMediaDownloader by clicking the 'browse' button above and the viewer will generate a twitter style interface for viewing the tweets.

Q & A

Is any data upload from my device?
No. The viewer runs completely within your web browser and does not collect or transmit any data.

Can I run it offline?
Yes. The viewer can be run offline and a single file version is available.

Does it use any third party JavaScript libraries?
Yes. The viewer uses two third party JavaScript libraries:

  • JSZip (v3.10.1) - JavaScript class for generating and reading zip files.
  • Papa Parse (v5.3.2) - Fastest in-browser CSV parser for JavaScript.
